#  Savings Bot — Твой личный финансовый помощник в Telegram

> Бот, который поможет копить деньги, следить за вкладами и никогда не забывать о своих сбережениях

## Что умеет этот бот?

Предоставляет возможность хранить в одном месте разнородные активы, учитывая проценты и срок.

### Основные возможности:

| Команда | Что делает | Пример |
|---------|-----------|--------|
| `/start` | Запускает бота и показывает главное меню | `/start` |
| `/new` | Создает новый вклад или кэш | `/new` |
| `/list` | Показывает все твои активы | `/list` |
| `/total` | Общий баланс по всем счетам | `/total` |
| `/add` | Пополняет существующий актив | `/add 50000 СберВклад` |
| `/take` | Снимает деньги с актива | `/take 10000 Заначка` |
| `/del` | Удаляет актив | `/del СберВклад` |
| `/help` | Показывает подсказки | `/help` |

### Типы активов:

- **Вклад (deposit)** — деньги лежат под проценты, бот сам начисляет доход каждый день
- **Наличные (cash)** — просто копилка без процентов

---

## Как начать пользоваться?

### Для пользователя (просто и быстро):

1. Найди бота в Telegram: **@ohmy_savings_bot**
2. Нажми `/start`
3. Создай первый актив через кнопку или команду `/new`
4. Указывай все пополнения и снятия — бот всё посчитает сам!

### Для разработчика (запустить своего бота):

#### Что нужно:
- Python 3.9 или новее
- Аккаунт в Telegram
- Токен бота от [@BotFather](https://t.me/BotFather)

#### Пошаговая установка:

```bash
# 1. Скачай проект
git clone https://github.com/твой-ник/savings-telegram-bot.git
cd savings-telegram-bot

# 2. Создай виртуальное окружение (рекомендуется)
python -m venv venv
source venv/bin/activate  # на Linux/Mac
venv\Scripts\activate     # на Windows

# 3. Установи зависимости
pip install -r requirements.txt

# 4. Создай файл с настройками
cp config_example.py config.py
# Открой config.py и вставь свой токен от @BotFather

# 5. Запусти бота
python bot.py
